Predicates

Up 
get /predicates
Get supported relationships by source and target (predicatesGet)

Return type

map[String, map[String, array[String]]]

Example data

Content-Type: application/json
{
  "chemical_substance" : {
    "gene" : [ "directly_interacts_with", "decreases_activity_of" ]
  }
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Edge Up

An edge in the thought subgraph linking two nodes

EdgeBinding Up

qg_id
String Query-graph edge id, i.e. the <code>edge_id</code> of a QEdge
kg_id
One or more knowledge-graph edge ids, i.e. the <code>id</code> of a KEdge

KnowledgeGraph Up

A thought graph associated with this result. This will commonly be a linear path subgraph from one concept to another, but related items aside of the path may be included.

Query Up

QueryGraph Up

A graph intended to be the thought path to be followed by a reasoner to answer the question. This graph is a representation of a question.

RemoteKnowledgeGraph Up

A thought graph associated with this result that is not repeated here, but stored elsewhere in a way that can be remotely accessed by the reader of this Message

String
